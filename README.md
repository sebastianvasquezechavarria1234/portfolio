### 🖋️ Editorial Excellence in Digital Form

This is a premium, editorial-inspired digital portfolio. Migrated from static code to a high-performance architecture using **React 19** and **Vite**, it focuses on advanced visual storytelling and an uncompromising user experience.

---

## ✨ Design Philosophy

The project follows a **high-end editorial aesthetic**, characterized by:
- **Bold Typography**: High-impact headings using `Bricolage Grotesque`.
- **Sophisticated Whitespace**: A clean, breathable layout that prioritizes content hierarchy.
- **Advanced Visuals**: Custom-engineered effects like the *Gradual Blur Header* and interactive *Bento Grids*.

---

## 🚀 Key Features

- **Fluid Navigation**: Powered by [Lenis Smooth Scroll](https://github.com/darkroomengineering/lenis) for a premium browsing feel.
- **Gradual Blur Header**: An Apple-inspired progressive blur effect that adds depth and elegance.
- **Bento Grid Layout**: Dynamic and interactive sections for Services and the Lab.
- **High Performance**: Optimized with Vite for near-instant loading and smooth transitions.
- **Responsive Architecture**: Pixel-perfect design that adapts seamlessly from ultra-wide displays to mobile devices.

---

## 🛠️ Tech Stack

### Frontend Core
- **React 19**: Leveraging the latest features for modern web development.
- **Vite 8**: Next-generation frontend tooling for a fast development cycle.
- **Tailwind CSS 4**: Utilizing the latest utility-first styling patterns.

### Motion & UX
- **GSAP**: Industry-standard animation library for complex, high-performance motion.
- **Lenis**: Smooth scrolling for a refined user interaction.
- **OverlayScrollbars**: Customized, elegant scrollbar experience.

## 📂 Project Structure

```bash
src/
├── components/   # Modular UI components (Hero, About, Bento Grids...)
├── assets/       # Static resources and brand assets
├── context/      # Global state management
├── index.css     # Global styles and Tailwind configuration
└── App.jsx       # Application entry point and layout
```

## 🏗️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/sebastianvasquezechavarria1234/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
